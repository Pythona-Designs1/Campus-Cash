# 💰 CampusCash — Campus Mini Budget Tracker

> **University of Ilorin | Faculty of Communication and Information Science | COS 102**

---

## 📌 About The Project

CampusCash is a Python-based budget tracking application built as a COS 102 group project. It helps students track their income and expenses, calculate their balance, and stay within budget.

The project comes in two versions:
- **GUI Version** (`CampusCash.py`) — A graphical window application built with Python's tkinter library
- **CLI Version** (`budget_tracker_cli.py`) — A terminal-based version for users without a graphical interface

---

## ✨ Features

- ➕ Add income entries with description and amount
- ➖ Add expense entries with description and amount
- 📊 Automatically calculates total income, total expenses and balance
- 🗑️ Delete any record from the tracker
- ✅ Budget status indicator (Within budget / Over budget / Balanced)
- 💻 GUI version with a beautiful dark-themed interface
- 📱 CLI version compatible with Android via Pydroid 3

---

## 🛠️ Technologies Used

- **Language:** Python 3.12
- **GUI Library:** tkinter (built into Python — no installation needed)
- **Interface:** Graphical User Interface (GUI) + Command Line Interface (CLI)

---

## 🚀 How To Run

### GUI Version (Laptop)
```bash
python CampusCash.py
```

### CLI Version (Terminal or Phone)
```bash
python budget_tracker_cli.py
```

> **Note:** Python 3.12 must be installed on your device. tkinter comes built into Python — no extra installation needed.
